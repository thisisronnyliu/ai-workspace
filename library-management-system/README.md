# 图书管理系统

一个美观、功能齐全的纯前端图书管理系统。

## 功能特性

- 📚 **图书管理**：添加、编辑、删除图书
- 🔍 **搜索功能**：按书名、作者、ISBN或分类搜索
- 🏷️ **分类管理**：支持多种图书分类
- 💼 **借阅管理**：标记图书为借出或归还状态
- 📊 **统计信息**：显示总图书数、可借阅数、已借出数
- 📁 **数据导出**：导出图书数据为JSON文件
- 💾 **本地存储**：数据持久化保存在浏览器中
- 📱 **响应式设计**：支持桌面和移动设备
- ✨ **现代UI**：美观的渐变色设计和流畅动画

## 技术栈

- HTML5
- CSS3 (包含Flexbox/Grid、渐变、动画)
- JavaScript (ES6+)
- localStorage (数据持久化)

## 使用方法

1. 直接打开 `index.html` 文件即可使用
2. 所有数据都保存在浏览器的本地存储中

## 系统要求

支持所有现代浏览器（Chrome, Firefox, Safari, Edge）
