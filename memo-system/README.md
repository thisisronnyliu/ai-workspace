# 备忘录管理系统

一个功能丰富、界面美观的备忘录管理应用，支持多种实用功能，帮助用户高效管理日常任务和提醒。

## 功能特色

### 核心功能
- **添加备忘录** - 支持快速添加新的备忘录条目
- **标记完成** - 点击复选框标记任务完成/未完成状态
- **编辑备忘录** - 支持修改已有备忘录内容
- **删除备忘录** - 单个删除不需要的备忘录
- **过滤显示** - 按状态筛选（全部/未完成/已完成）

### 高级功能
- **本地存储** - 所有数据保存在浏览器本地存储中，刷新不丢失
- **统计信息** - 实时显示备忘录统计（总计/未完成/已完成）
- **批量操作** - 一键清除所有已完成的备忘录
- **数据导出** - 支持将所有备忘录导出为JSON格式文件
- **响应式设计** - 适配桌面端和移动端设备

### 用户体验
- **流畅动画** - 平滑的过渡动画提升用户体验
- **键盘快捷键** - 支持回车键快速添加备忘录
- **直观界面** - 清晰的视觉层次和色彩搭配
- **操作确认** - 删除操作有确认提示，防止误删

## 技术特点

- 使用原生 JavaScript 开发，无依赖库
- 响应式 CSS 设计，兼容各种设备
- 本地存储机制，数据持久化
- 模块化代码结构，易于维护扩展

## 使用说明

1. 在顶部输入框中输入备忘录内容
2. 点击"添加备忘录"按钮或按回车键添加
3. 点击复选框标记任务完成状态
4. 使用编辑/删除按钮管理单个备忘录
5. 通过过滤按钮切换不同视图
6. 使用底部操作按钮进行批量处理

## 数据安全

- 所有数据仅保存在本地浏览器中
- 不涉及任何网络传输或云端存储
- 用户完全掌控自己的数据
- 支持数据导出备份功能
